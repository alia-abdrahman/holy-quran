# T-008: Create a specific favourite page to locate all the favourited surah and ayah by user

**Epic:** EPIC-009
**Status:** Done

## Description
1. User should found a heart icon on the right side of bookmark icon on the surah listing page - Refer to the image attached
If the user click the heart icon, user will be redirected to the favourite page
![favourite_page](resources/images/favourite_page.png)
2. The page should contain both favourited surah and favourited ayah section
3. User should be able to go to the favourited surah/ayah upon user click from the section made


## Acceptance Criteria
- [x] Heart icon visible on HomePage header (next to search icon)
- [x] Clicking heart icon navigates to FavouritePage
- [x] FavouritePage displays Favourite Surahs section
- [x] FavouritePage displays Favourite Ayahs section
- [x] Clicking a favourite surah navigates to SurahDetailPage
- [x] Clicking a favourite ayah navigates to SurahDetailPage
- [x] Users can remove favourites directly from FavouritePage

## Technical Notes
- Created `lib/pages/favourite_page.dart` - new page to display all favourites
- Modified `lib/pages/home_page.dart` - added heart icon with navigation to FavouritePage
- Utilizes existing `FavouriteService` for persisting favourites via SharedPreferences
- FavouritePage refreshes when returning from SurahDetailPage to reflect any changes

## Dependencies

- T-007 (Ayah favourite functionality) - Done

## Testing
- Favourite a surah from SurahDetailPage
- Favourite an ayah from SurahDetailPage
- Navigate to FavouritePage via heart icon on HomePage
