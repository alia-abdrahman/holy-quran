# T-006: User can favourite any surah by click on the heart icon located beside the bookmark icon

**Epic:** EPIC-009
**Status:** Done

## Description
- User can click on the favourite icon which should be located on the left side of the bookmark icon
![favourite_icon_for_surah](resources/images/favourite_icon_for_surah.png)

- [x] Heart icon displayed to the left of bookmark icon in SurahDetailPage app bar

- [x] Favourite state persists across app restarts
- [x] Active favourite shows pink color (#D946A1)

- Uses `shared_preferences` for persistence
- Favourite surah stored as list of surah numbers

## Dependencies
- shared_preferences package added to pubspec.yaml

## Testing

- Tap the heart icon to favourite
- Icon should turn pink and filled
