# T-002: User can search surah by typing the name of surah

**Epic:** EPIC-008
**Status:** Done

## Description
- While user type surah name, show live suggestion based on user input

## Acceptance Criteria
- [x] User can tap search icon to open search field
- [x] Live filtering as user types
- [x] Search matches surah English name, translation, number, and Arabic name
- [x] "No results" state shown when no matches
- [x] User can close search to return to full list

## Technical Notes
- Implemented in `lib/pages/home_page.dart`
- Uses TextEditingController with listener for live updates
- Client-side filtering (no API calls needed)
- Matches existing app theme and design patterns

## Dependencies
- None

## Testing
- Tap search icon in header to open search field
- Type "fatih" to see Al-Fatihah filtered
- Type "cow" to see Al-Baqarah (The Cow) filtered
- Type "2" to see surahs with "2" in their number
- Type invalid text to see "No surah found" message
- Tap X icon to close search and return to full list
