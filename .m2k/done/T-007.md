# T-007: User can favourite any ayah in a surah by click on the heart icon located beside the bookmark icon

**Epic:** EPIC-009
**Status:** Done

## Description
- User can click on the favourite icon which should be located on the left side of the bookmark icon
![favourite_icon_for_ayah](resources/images/favourite_icon_for_ayah.png)

## Acceptance Criteria
- [x] Heart icon displayed to the left of bookmark icon on each ayah card
- [x] Heart icon toggles between outline and filled when tapped
- [x] Favourite state persists across app restarts
- [x] Active favourite shows pink color (#D946A1)

## Technical Notes
- Uses same `FavouriteService` from T-006
- Ayah favourites stored as "surahNumber:ayahNumber" format
- Updated `_buildActionIcon` to support active state styling

## Dependencies
- T-006 (FavouriteService)

## Testing
- Open any surah detail page
- Tap the heart icon on any ayah to favourite
- Icon should turn pink and filled
- Close and reopen app - favourites should persist
